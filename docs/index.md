# Project Documentation Index

## Project Overview
- **Name:** MoneyShyft
- **Type:** monorepo with 2 parts
- **Primary Language:** TypeScript
- **Architecture:** Vue 3 SPA + Express API

## Quick Reference

### frontend (web)
- **Type:** Web
- **Tech Stack:** Vue 3, Vite, Pinia, Tailwind, TypeScript
- **Root:** `frontend/`

### src (backend)
- **Type:** Backend
- **Tech Stack:** Node.js, Express, Knex, PostgreSQL
- **Root:** `src/`
